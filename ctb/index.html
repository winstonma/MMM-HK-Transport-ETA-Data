HK Transport ETA Data

This site provides real-time data for Citybus (CTB) routes and stops in Hong Kong, updated daily.

Data Access

All data is available as JSON files in the stops and routes directories:

Individual Stops

Format: https://[username].github.io/MMM-HK-Transport-ETA-Data/stops/[stop_id].json

Example: stops/001001.json

All Stops

Complete dataset: stops/allstops.json

Individual Routes

Format: https://[username].github.io/MMM-HK-Transport-ETA-Data/routes/[route].json

Example: routes/1.json

All Routes

Complete dataset: routes/allroutes.json

Data Structure

Stop Files

Each stop JSON file contains:

  • stop - Stop ID
  • name_tc - Traditional Chinese name
  • name_en - English name
  • lat, long - Coordinates
  • name_sc - Simplified Chinese name
  • data_timestamp - When the data was collected
  • routes - Array of route numbers that serve this stop

Route Files

Each route JSON file contains:

  • route - Route ID
  • stops - Array of all stops (both inbound and outbound) with full details

Each stop in the route files includes:

  • co - Company code (CTB)
  • route - Route ID
  • dir - Direction (I = Inbound, O = Outbound)
  • seq - Sequence number
  • stop - Stop ID
  • data_timestamp - When the data was collected
  • name_tc - Traditional Chinese name
  • name_en - English name
  • name_sc - Simplified Chinese name
  • lat, long - Coordinates

Usage

This data can be used for:

  • Developing transport apps
  • Creating visualizations
  • Research and analysis
  • Magic Mirror modules (like MMM-HK-Transport-ETA)

API Information

The data is collected daily at 5 AM Hong Kong time. The dataset includes:

  • All CTB bus stops in Hong Kong
  • Geographic coordinates for each stop
  • Route information for each stop

Source

Data is collected from the Hong Kong Government's Transport Real-Time Information API.
