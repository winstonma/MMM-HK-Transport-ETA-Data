HK Transport ETA Data

This site provides real-time data for Citybus (CTB) routes and stops in Hong Kong, updated daily.

Data Access

All data is available as JSON files. Each stop has its own file named with its stop ID:

Format: https://[username].github.io/MMM-HK-Transport-ETA-Data/[stop_id].json

Example: 001001.json

Data Structure

Each JSON file contains:

  • stop - Stop ID
  • name_tc - Traditional Chinese name
  • name_en - English name
  • lat, long - Coordinates
  • name_sc - Simplified Chinese name
  • data_timestamp - When the data was collected
  • routes - Array of route numbers that serve this stop

Usage

This data can be used for:

  • Developing transport apps
  • Creating visualizations
  • Research and analysis
  • Magic Mirror modules (like MMM-HK-Transport-ETA)

Source

Data is collected from the Hong Kong Government's Transport Real-Time Information API.
